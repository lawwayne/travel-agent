# Travel Planning Multi-Agent System

An AI-powered travel planning system built with Google's Agent Development Kit (ADK). Features specialized agents for flights, hotels, and attractions working together.

## 🏗️ Architecture

- **Flight Specialist**: Handles flight bookings and inquiries
- **Hotel Specialist**: Manages accommodation searches  
- **Attraction Specialist**: Recommends local activities
- **Coordinator Agent**: Routes complex requests to specialists

## 🚀 Quick Start

1. Install dependencies: `pip install google-adk`
2. Add Gemini API key to `.env`
3. Run: `adk web`

## 🛠️ Tools Used

- Google ADK (Agent Development Kit)
- Gemini 2.5 Flash (free tier)
- Python 3.x
